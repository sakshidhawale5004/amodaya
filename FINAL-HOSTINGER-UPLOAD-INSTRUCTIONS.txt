🔧 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════

PROBLEM: Certificates show 404 error
SOLUTION: 
- Check that "certificates" folder is lowercase
- Check that it's directly in public_html (not nested)
- Check that files are directly inside certificates/ (no subfolders)

PROBLEM: Blank page or CSS not loading
SOLUTION:
- Clear browser cache (Ctrl + Shift + Delete)
- Check that index.html is in public_html root
- Check that assets/ folder exists

PROBLEM: Page refresh shows 404
SOLUTION:
- Check that .htaccess file exists in public_html
- Contact Hostinger support to enable mod_rewrite

PROBLEM: Old content still showing
SOLUTION:
- Clear browser cache completely
- Try incognito/private browsing mode
- Wait 5-10 minutes for server cache to clear
