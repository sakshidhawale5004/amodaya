================================================================================
                        TROUBLESHOOTING
================================================================================

If you see 404 errors:

1. Check .htaccess file
   - Must be in public_html/ root (not in subfolder)
   - File name is .htaccess (with dot, no extension)
   - Check file permissions (644 or 755)

2. Clear browser cache
   - Ctrl+Shift+Delete (Windows)
   - Cmd+Shift+Delete (Mac)
   - Select "All time"

3. Hard refresh
   - Ctrl+F5 (Windows)
   - Cmd+Shift+R (Mac)

4. Try incognito window
   - Ctrl+Shift+N (Windows)
   - Cmd+Shift+N (Mac)
   - Fresh cache, fresh session

5. Check file structure on Hostinger
   - public_html/.htaccess (must exist)
   - public_html/index.html (must exist)
   - public_html/assets/ (must exist)
