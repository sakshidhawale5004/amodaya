                          TROUBLESHOOTING QUICK REF
================================================================================

IF FAVICON NOT UPDATING:
→ Hard refresh: Ctrl+Shift+Delete (Chrome) or Cmd+Shift+Delete (Safari)
→ Clear cache
→ Try incognito window
→ Wait 24 hours for Hostinger cache

IF CERTIFICATES 404:
→ Verify certificates/ folder exists (lowercase)
→ Check all 14 files uploaded
→ Verify file names exact match
→ Check folder permissions

IF IMAGES NOT LOADING:
→ Verify assets/ folder exists
→ Hard refresh: Ctrl+F5
→ Check file permissions
→ Clear cache

IF ROUTING NOT WORKING:
→ Verify .htaccess in public_html/
→ Check mod_rewrite enabled
→ Contact Hostinger support
